# Трекер прочитанных книг

Консольное приложение на Python для хранения списка прочитанных книг.

Приложение позволяет:

- добавлять книгу;
- показывать список всех книг;
- считать среднюю оценку;
- показывать статистику по авторам;
- удалять книгу из списка.

Данные хранятся в файле `books.json`.

## Поля книги

У каждой книги есть:

- автор;
- название;
- оценка от 1 до 5;
- дата прочтения в формате `ГГГГ-ММ-ДД`.

## Как запустить

1. Убедитесь, что установлен Python.
2. Скачайте или клонируйте репозиторий.
3. Откройте папку проекта в терминале.
4. Запустите команду:

```bash
python main.py
```

Если команда не работает, попробуйте:

```bash
python3 main.py
```

## Работа с Git и GitHub

Проект выполнялся как имитация командной разработки.

Использованные ветки:

- `feature/add-book` — добавление книги и проверка дубликатов;
- `feature/list-and-stats` — вывод списка, средняя оценка, статистика по авторам;
- `feature/delete` — удаление книги.

Минимум одна ветка была объединена с `main` через Pull Request.
